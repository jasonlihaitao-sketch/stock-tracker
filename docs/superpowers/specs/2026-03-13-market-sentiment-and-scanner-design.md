# 市场情绪看板与全市场扫描器设计文档

## 概述

为股票跟踪工具新增两个核心功能，帮助用户主动发现投资机会：

1. **市场情绪看板** - 实时展示市场整体状态，辅助判断操作时机
2. **全市场扫描器** - 扫描全市场股票，发现符合买入策略的机会

## 功能设计

### 一、市场情绪看板

#### 位置
首页顶部，自选股列表上方

#### 布局
```
┌─────────────────────────────────────────────────────────────┐
│  市场情绪看板（单行卡片）                                     │
│  ┌─────────┬─────────┬─────────┬─────────┬─────────────┐   │
│  │ 上证指数 │ 深证成指 │ 创业板指 │ 涨跌比  │ 资金净流入   │   │
│  │ 3XXX.XX │ 1XXXX.X │ 2XXX.XX │ 35:65   │ -125亿      │   │
│  │ +0.5%   │ -0.3%   │ +1.2%   │ 📊弱    │ 💰流出      │   │
│  └─────────┴─────────┴─────────┴─────────┴─────────────┘   │
├─────────────────────────────────────────────────────────────┤
│  热门板块 TOP3（紧凑展示）                                    │
│  🔥 半导体 +2.3% | 人工智能 +1.8% | 新能源 +1.5%             │
└─────────────────────────────────────────────────────────────┘
```

#### 组件说明

| 组件 | 数据内容 | 更新频率 | 数据源 |
|------|----------|----------|--------|
| 大盘指数卡片 | 上证/深证/创业指数、涨跌幅 | 3秒 | 新浪API |
| 涨跌比卡片 | 上涨家数:下跌家数、强弱标签 | 30秒 | 东财API |
| 资金流向卡片 | 主力净流入金额、流向标签 | 30秒 | 东财API |
| 热门板块 | 板块名称+涨跌幅 TOP3 | 5分钟 | 东财API（复用现有） |

#### 强弱判断逻辑

| 指标 | 强 | 中 | 弱 |
|------|----|----|-----|
| 涨跌比 | > 60%上涨 | 40-60%上涨 | < 40%上涨 |
| 资金流向 | 流入 > 50亿 | -50~50亿 | 流出 > 50亿 |

```
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

#### 扫描范围选项

| 范围 | 数量 | 预计时间 |
|------|------|----------|
| 全市场 | ~5000只 | 5-10分钟 |
| 自选股 | 用户自选股数量 | 几秒 |
| 沪深300 | 300只 | 30秒 |

#### 扫描策略（复用现有）

| 策略 | 检测条件 | 信号类型 |
|------|----------|----------|
| 突破策略 | 当前价 > 20日最高价 | 突破20日高 |
| 量价策略 | 成交量 > 5日均量 × 2 | 量价配合 |
| 板块共振 | 个股涨 + 所属板块涨 > 1% | 板块共振 |

#### 信号强度计算

- 单一信号：⭐⭐⭐
- 两个信号：⭐⭐⭐⭐
- 三个信号：⭐⭐⭐⭐⭐

#### 结果列表字段

| 字段 | 说明 |
|------|------|
| 代码/名称 | 股票基本信息，可点击跳转详情页 |
| 现价 | 当前价格 |
| 涨跌幅 | 今日涨跌幅，带颜色 |
| 信号类型 | 突破/量价/板块共振，多个用逗号分隔 |
| 信号强度 | 1-5星显示 |
| 操作 | 一键加入自选 |

#### 扫描流程

```
1. 用户点击「开始扫描」
      ↓
2. 获取股票列表（剔除ST、停牌、上市不足60天）
      ↓
3. 分批请求（每批50只，间隔500ms）
      ↓
4. 对每只股票检测信号
      ↓
5. 有信号的股票加入结果列表（增量更新UI）
      ↓
6. 按信号强度排序展示
```

#### 性能优化策略

| 策略 | 说明 |
|------|------|
| 分批请求 | 每批50只，间隔500ms，避免API限制 |
| 增量更新 | 每发现信号立即更新UI |
| 可中断 | 支持中途停止扫描 |
| 结果缓存 | 结果保存到localStorage |

### API 接口设计

#### 市场统计数据

```typescript
// lib/api/market.ts
interface MarketStats {
  upCount: number      // 上涨家数
  downCount: number    // 下跌家数
  limitUp: number      // 涨停数
  limitDown: number    // 跌停数
  netInflow: number    // 主力净流入（亿元）
}

function getMarketStats(): Promise<MarketStats>
```

#### 股票列表

```typescript
// lib/scanner/stock-list.ts
function getScanableStocks(): Promise<string[]>
// 返回可扫描股票代码列表（剔除ST、停牌、上市不足60天）
```

#### 扫描器

```typescript
// lib/scanner/index.ts
interface ScanOptions {
  scope: 'all' | 'watchlist' | 'hs300'
  onProgress?: (current: number, total: number) => void
  onFound?: (result: ScanResult) => void
}

interface ScanResult {
  code: string
  name: string
  price: number
  changePercent: number
  signals: ('breakout' | 'volume' | 'sector')[]
  strength: number  // 1-5
}

function startScan(options: ScanOptions): Promise<ScanResult[]>
function stopScan(): void
```

### 状态管理

```typescript
// store/scanStore.ts
interface ScanState {
  status: 'idle' | 'scanning' | 'completed'
  progress: { current: number; total: number }
  results: ScanResult[]
  scope: 'all' | 'watchlist' | 'hs300'

  startScan: (scope: string) => void
  stopScan: () => void
  clearResults: () => void
}
```
