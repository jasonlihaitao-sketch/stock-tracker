## 一、文档概述

### 1.1 目的

本文档定义了股票跟踪网站所有功能模块的测试计划，包括测试范围、测试用例、验收标准和测试策略。

### 1.2 测试范围

| 模块 | 页面路径 | 测试优先级 |
|------|----------|------------|
| 自选股管理 | `/` | P0 |
| 股票详情 | `/stock/[code]` | P0 |
| 持仓管理 | `/portfolio`, `/position` | P0 |
| 预警系统 | `/alerts` | P0 |
| 市场扫描 | `/scan` | P1 |
| 策略雷达 | `/strategy` | P1 |
| 股票筛选 | `/screener` | P1 |
| 板块雷达 | `/sector` | P1 |
| 操作计划 | `/plans` | P1 |
| AI助手 | `/ai-assistant` | P2 |

### 1.3 测试类型

| 类型 | 工具 | 覆盖目标 |
|------|------|----------|
| E2E测试 | Playwright | 核心用户流程 |
| 单元测试 | Vitest/Jest | 工具函数、Store逻辑 |
| 集成测试 | Playwright | 组件交互、API调用 |
| 手动测试 | - | UI/UX、边界场景 |

#### 数据验证点

```
- 买入价: buyPrice > 0
- 数量: quantity > 0, 必须是100的整数倍（A股）
- 盈亏计算: profit = (currentPrice - buyPrice) * quantity
- 收益率: profitPercent = (currentPrice - buyPrice) / buyPrice * 100
- 移动止损: 当前价创新高后更新止损价
```
